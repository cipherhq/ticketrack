# Comprehensive Feature List - Ticketrack

This document lists all features available in Ticketrack for reference when updating the Help Center.

## Attendee Features
- Browse events with filters (date, price, category, location)
- Search events
- Location-based event sorting (nearest first)
- View event details
- Purchase tickets (single or multiple types)
- Free event RSVP
- Donation tickets
- Recurring event date selection
- Waitlist for sold-out events
- My Tickets dashboard
- Ticket QR codes
- Download tickets as PDF
- Add tickets to Apple/Google Wallet
- Request refunds
- Transfer tickets
- View transferred tickets history
- Follow organizers
- Save favorite events
- Calendar integration
- Payment methods: Card, Bank Transfer, USSD, Mobile Money, PayPal

## Organizer Features

### Event Creation & Management
- Create single events
- Create recurring events (daily, weekly, biweekly, monthly, custom)
- Create multi-day events
- Virtual events with streaming links
- Free events with RSVP
- Donation tickets
- Event visibility: Public, Unlisted, Password-protected, Invite-only
- Scheduled publishing
- Custom checkout forms
- Custom event URLs
- Event banners and images
- Venue layout designer (2D floor plans)
- Venue management
- Location-based filtering (US users see only US events)
- Auto-refund on event cancellation

### Ticketing
- Multiple ticket types per event
- Table/group tickets
- Quantity limits per ticket type
- Max tickets per order
- Ticket transfer settings (enable/disable, fees)
- Manual ticket issuance (complimentary or paid)
- Ticket inventory per recurring event date

### Promotions & Marketing
- Promo codes (percentage or fixed discount)
- Promoter program with commission tracking
- Email campaigns
- SMS campaigns
- WhatsApp broadcasts (via 360dialog, WATI, Twilio, Termii)
- Follower system
- Social sharing

### Sales & Attendees
- Orders management
- Attendee list with custom form responses
- Export attendees to CSV
- Check-in by QR code or manual search
- Transfer tracking
- Post-event dashboard

### Finance
- Payouts (Stripe Connect, Paystack subaccounts, Flutterwave)
- Bank account management
- KYC verification
- Tax documents
- Analytics dashboard
- Revenue tracking by currency
- Refund management

### Team & Projects
- Team member management
- Project manager (Kanban board)
- Task assignment and tracking

### Settings
- Profile management
- Bank account setup
- Stripe Connect integration
- KYC verification
- Tax document submission

## Platform Features
- Multi-currency support (NGN, GHS, KES, ZAR, USD, GBP, EUR)
- Multiple payment gateways (Paystack, Stripe, PayPal, Flutterwave)
- Subaccounts for direct payouts
- Location-based content filtering
- Real-time notifications
- Email notifications
- SMS notifications (Twilio, Termii)
- Error tracking (Sentry)
